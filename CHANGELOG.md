# Changelog

All notable changes to PY_Code Flutter CLI will be documented in this file.

## [1.0.0] - 2026-04-19

### Added
- Initial release of PY_Code Flutter CLI
- Cloud build support via GitHub Actions
- Package manager with 8+ built-in installers
- Flutter SDK installation support
- Qwen AI assistant integration
- Shell tools bundle (Fish, Zsh, Oh-My-Zsh, TheFuck)
- Auto-update functionality
- Beautiful terminal UI with colors
- Full Termux compatibility

### Installers Included
- flutter - Flutter SDK
- qwen - Qwen Code CLI
- oh-my-zsh - Zsh + Oh My Zsh + Plugins
- shell-tools - Fish, FZF, Zsh, TheFuck bundle
- neovim - Modern Vim editor
- micro - Simple terminal editor
- fish - Friendly Interactive SHell
- zsh - Z Shell
- thefuck - Command correction tool
